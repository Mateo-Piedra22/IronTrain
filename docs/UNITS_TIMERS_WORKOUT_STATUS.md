# Unidades, timers y estado de entrenamiento

## Unidades

- Soporte para peso/repeticiones con normalización interna.
- Consistencia de unidades crítica para analíticas y comparaciones.

## Timers

- Rest timer configurable.
- Integración con flujo de sets y overlays de entrenamiento.

## Estado de workout

- Registro progresivo por set/día/rutina.
- Persistencia local-first en SQLite.
- Eventos de actualización para sincronizar vistas activas.
